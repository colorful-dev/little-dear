# 小鹿记账

## 如何启动项目
```shell
git clone git@github.com:colorful-dev/little-dear.git
cd little-dear
pnpm install
cp .env.example .env
## 然后编辑.env 文件 修改username 和password
docker-compose up -d
pnpm dev
```
